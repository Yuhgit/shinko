## 画像の取得元

すべての画像はFigmaデザインファイルから自動的にエクスポートされ、このディレクトリに配置されました。

## 注意事項

- 画像ファイル名はコード内で指定された名前と完全に一致しています
- SVGファイルはベクター形式でエクスポートされています
- PNG/AVIF/JPGファイルは適切な解像度でエクスポートされています
- 画像ファイルのパスは `/images/` から始まります（例: `/images/site-title.png`）
