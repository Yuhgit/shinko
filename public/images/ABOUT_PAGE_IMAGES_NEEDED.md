# 会社概要ページで必要な画像ファイル

以下の画像ファイルをFigmaからエクスポートして、`public/images/`フォルダに配置してください。

### ページヘッダー
- `about-header-bg.png` (1440x508px) - ページヘッダーの背景画像

### 代表挨拶セクション
- `ceo-image.png` (472x368px) - 代表取締役の写真

### 特徴カード
- `rectangle-12.png` (275x352px) - FEATURE 01の画像
- `rectangle-13.png` (275x446px) - FEATURE 02の画像
- `rectangle-14.png` (275x446px) - FEATURE 03の画像

### 会社概要セクション
- `company-image-1.png` (900x630px) - 会社外観画像
- `company-image-2.png` (900x630px) - 会社内観画像

### 認証・ライセンス
- `youth-ell.png` (130x126px) - ユースエール認定ロゴ
- `kawasaki-ell.png` (160x146px) - かわさき☆えるぼし認証ロゴ
- `sdgs.png` (212x109px) - SDGs認証ロゴ
- `iso-anab.png` (323x314px) - ISO ANAB認証ロゴ
- `iso-eqa.png` (178x87px) - ISO EQA認証ロゴ

### アクセス
- `map.png` (1200x400px) - アクセス地図

## 既に作成済みのSVGアイコン
- ✅ `chevron-right.svg` - パンくずリストの矢印
- ✅ `section-divider-orange.svg` - オレンジのセクション区切り線
- ✅ `section-divider-white.svg` - 白のセクション区切り線

## Figmaからのエクスポート方法
1. Figmaで該当のノードを選択
2. 右クリック → "Export" を選択
3. 適切な形式（PNG推奨）とサイズを選択
4. `public/images/`フォルダに保存

## 一時的なプレースホルダー
画像が準備できるまで、既存の画像を一時的に使用することもできます：
- `about-header-bg.png` → `contact-bg.png`を一時的に使用可能
- `ceo-image.png` → `rectangle-3.png`を一時的に使用可能
- `rectangle-12.png`, `rectangle-13.png`, `rectangle-14.png` → `rectangle-5.png`, `rectangle-6.png`を一時的に使用可能
